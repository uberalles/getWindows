getWindows 1.0
========

**Free, Open Source light opened Windows Manager**

getWindows 1.0 is coded in C# (using .NET Framework 4.5).

Requirements
---
* .NET Framework 4.5

License
---
GPLv3, see LICENSE file for details.

Features
---
+ Maximize Window
+ Minimize Window
+ Hide Window
+ Show Window in normal state
